Fast teacher workflow

  1. Choose the session hub.
  2. Find the category that matches the teacher need.
  3. Open the small explanation popup if the tool/resource is unfamiliar.
  4. Copy the prompt or send it to the workspace.
  5. Adapt it, run the F.A.C.T. check, and paste into Google Docs if needed.

Safety anchor

Barrier removed. Thinking preserved. Learning target unchanged.

Teachers should never enter student names, IDs, exact grades, IEP/504 details, or identifying stories into AI tools. Use class-level patterns and anonymized examples only.
